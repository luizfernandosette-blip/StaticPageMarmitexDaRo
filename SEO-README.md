# Otimizações de SEO Implementadas

Este site foi otimizado para aparecer nas buscas locais do Google quando pessoas procurarem por:
- Marmitex em Sabará
- Tele-entrega Sabará
- Comida caseira Sabará
- Delivery Sabará
- Buffet Sabará
- E outras variações relacionadas

## O que foi implementado:

### 1. Meta Tags Otimizadas
- Title tag com palavras-chave principais
- Meta description detalhada
- Keywords locais relevantes
- Geo tags para localização (Sabará, MG)
- Open Graph para compartilhamento em redes sociais

### 2. Dados Estruturados (Schema.org)
- JSON-LD com informações do restaurante
- Dados de localização (endereço, telefone)
- Horário de funcionamento
- Cardápio estruturado
- Links para redes sociais

### 3. Conteúdo Otimizado
- Título H1 com palavras-chave principais
- Alt texts das imagens com termos de busca locais
- Seção informativa com conteúdo rico em palavras-chave
- Menções a bairros e região de atendimento

### 4. Arquivos Técnicos
- **robots.txt**: Permite indexação pelos buscadores
- **sitemap.xml**: Mapa do site para Google
- **.htaccess**: Otimizações de cache e performance
- Canonical URL definida

## Próximos Passos Recomendados:

### 1. Google My Business (ESSENCIAL)
Cadastre o negócio no Google Meu Negócio:
- Acesse: https://business.google.com
- Adicione fotos do estabelecimento
- Complete todas as informações
- Verifique o endereço
- Peça avaliações de clientes

### 2. Google Search Console
- Cadastre o site: https://search.google.com/search-console
- Envie o sitemap.xml
- Monitore erros e performance

### 3. Redes Sociais
- Mantenha o Instagram ativo e atualizado
- Use hashtags locais: #SabaráMG #MarmitexSabará
- Poste fotos dos pratos regularmente

### 4. Avaliações
- Incentive clientes a deixarem avaliações no Google
- Quanto mais avaliações positivas, melhor o ranking local

### 5. Conteúdo Regular
- Atualize o cardápio quando houver mudanças
- Adicione novos pratos com descrições detalhadas
- Mantenha a seção de vídeos atualizada

### 6. Velocidade do Site
- Otimize as imagens (reduza o tamanho sem perder qualidade)
- Use formato WebP quando possível
- Considere usar CDN para servir recursos estáticos
